nextcloud|php_fileinfo

发表于 2019-10-24 更新于 2021-06-08 分类于 all

虽然应该是很久之前就做过同样的事情，但是已经忘记了，所以还是得重新写下来。Nextcloud搬家完之后会推荐装fileinfo模块，如果用的是lnmp的一键包，就要去到src目录下面，把php的源代码包解压缩，然后cd进ext目录，就会看见fileinfo模块的源码目录，然后是：

/usr/local/php/bin/phpize
./configure –with-php-config=/usr/local/php/bin/php-config
make && make install

然后需要在php.ini里面把

extension=fileinfo.so

那一行取消注释。然后restart一下lnmp就可以了。
